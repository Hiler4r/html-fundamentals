fundamenals of css

CSS stands for Cascading Style Sheets. It is used to style and layout web pages. CSS allows you to control the color, font, spacing, and positioning of elements on a web page.

1. Selectors

Selectors are used to target HTML elements for styling. They define which elements the CSS rules apply to.

  • Element selector (e.g., p)
  • Class selector (e.g., .container)
  • ID selector (e.g., #header)
  • Attribute selector (e.g., [type="text"])
  • Descendant selector (e.g., nav ul)

2. Properties

Properties define the styles applied to selected elements, such as color, size, and spacing.

  • color
  • background-color
  • font-size
  • margin
  • padding

3. Layout

Layout branches control the positioning and arrangement of elements on the page.

  • display
  • position
  • float
  • flexbox
  • grid

4. Responsive Design

Responsive design ensures web pages look good on all devices by adapting styles based on screen size.

  • Media queries (@media)
  • Viewport units (vw, vh)
  • Flexible images
  • Fluid grids
  • Responsive typography

Summary

CSS branches include selectors, properties, layout, and responsive design. Each branch has specific features and techniques to enhance web page styling and usability.
